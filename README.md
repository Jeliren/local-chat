# Local Chat

Простой чат для локальной сети с отправкой сообщений и файлов. Работает через браузер и не требует внешних зависимостей: достаточно стандартного Python.

## Возможности

- общий чат для всех участников в одной сети;
- личные сообщения между пользователями;
- отправка файлов;
- вход по паролю перед доступом к переписке;
- запуск на одном компьютере и подключение с телефона, ноутбука или планшета;
- хранение истории сообщений и загруженных файлов на сервере.

## Быстрый старт

```bash
python3 app.py
```

По умолчанию сервер запускается на `0.0.0.0:8000`.
Если переменная `CHAT_PASSWORD` не задана, используется пароль `local-chat`.

Чтобы сразу задать свой пароль:

```bash
CHAT_PASSWORD="my-secret-pass" python3 app.py
```

После запуска откройте:

- на компьютере с сервером: `http://127.0.0.1:8000`
- на других устройствах в той же сети: `http://<локальный-ip-компьютера>:8000`

Приложение само показывает локальный адрес в интерфейсе и в консоли при запуске.

## Настройки

Можно изменить адрес и порт через переменные окружения:

```bash
CHAT_HOST=0.0.0.0 CHAT_PORT=9000 python3 app.py
```

## Где лежат данные

- сообщения: `data/messages.json`
- загруженные файлы: `data/uploads/`

## Если устройство в сети не открывает чат

- убедитесь, что оба устройства подключены к одной Wi-Fi сети;
- используйте локальный адрес вида `192.168.x.x`, а не VPN-адрес;
- проверьте, что firewall на компьютере не блокирует входящие подключения к Python.

## Идеи для развития

- отдельные комнаты;
- список пользователей онлайн;
- удаление сообщений;
- WebSocket вместо периодического опроса.
